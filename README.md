# Help Desk Simulation Project

## About This Project
This project simulates real-world IT help desk scenarios. It demonstrates how common technical issues are identified, diagnosed, and resolved.

---

## Ticket 1: WiFi Not Connecting

### Issue:
User reports that WiFi is not connecting on their laptop.

### Troubleshooting Steps:
1. Checked if WiFi is turned ON
2. Restarted router and laptop
3. Forgot and reconnected to network
4. Ran Network Troubleshooter

### Resolution:
WiFi connection restored after resetting network settings.

## Ticket 2: Slow Computer Performance

### Issue:
User reports system is very slow.

### Troubleshooting Steps:
1. Opened Task Manager
2. Identified high CPU usage applications
3. Closed unnecessary background apps
4. Restarted system

### Resolution:
System performance improved after closing high-usage applications.

### Screenshot:
![Slow Computer](slow-computer.png)

---

## Ticket 3: No Internet Access

### Issue:
User connected to WiFi but no internet access.

### Troubleshooting Steps:
1. Ran Command Prompt
2. Executed ipconfig
3. Checked IP address
4. Renewed IP using:
   - ipconfig /release
   - ipconfig /renew

### Resolution:
Internet restored after renewing IP address.

![IP Config](ipconfig-output.png)
